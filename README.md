# 修行地圖 · 三階道

依 淨界法師「三種道次第」編寫的佛學複習互動小遊戲。純前端、單檔 HTML，離線可玩。

## 道次第

安樂道（業感緣起 · 斷惡修善）→ 解脫道（賴耶緣起 · 破妄顯真）→ 菩提道（真如緣起 · 圓教）→ 會歸淨土

共 15 個節點、28 道選擇題，答案逐字源自淨界法師講記原文。

## 玩法

- 走完全程，**臨終時 願力 ＞ 業力** 即往生淨土
- 答對增願力、答錯增業力；**懺悔**節點答對可消業（全程唯一能降低業力之處）
- 答錯可念誦一遍、重答一次（每題限一次）

## 運行

用瀏覽器直接開啟 `修行地圖-三階道.html` 即可，無需安裝、無需聯網。

## 說明

道次第為教理**呈現**次第，非往生**條件**——淨土但須信願（願力強過業力）即得往生，圓教之理可早熏習，不必待修滿前階。本遊戲僅作複習助道，非教理定論。
